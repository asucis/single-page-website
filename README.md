# Single page website

An experiment to condense the usual website in a single page layout.

## 🚀 Quick Start

```sh
$ git clone https://github.com/asucis/single-page-website.git
$ cd single-page-website
$ yarn
$ yarn start
```

ℹ️ starts the website at localhost:3000
